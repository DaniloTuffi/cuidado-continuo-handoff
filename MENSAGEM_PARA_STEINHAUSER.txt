  LEFT JOIN unit_prices up ON up.package_id = cp.package_id AND up.unit_id = cp.unit_id

ETAPA 4 · Deploy da edge function issue-painel-jwt
─────────────────────────────────────────────────────────────────
  cd <repo-do-app>
  cp -r ../cuidado-continuo-handoff/edge-functions/issue-painel-jwt \
        supabase/functions/

  # Gera secret 32 bytes random
  PAINEL_JWT_SECRET=$(openssl rand -base64 32)
  echo "$PAINEL_JWT_SECRET"     # ← anota, vai precisar no Flask também
  supabase secrets set PAINEL_JWT_SECRET="$PAINEL_JWT_SECRET"

  supabase functions deploy issue-painel-jwt

Testa:
  curl -X POST 'https://<ref>.supabase.co/functions/v1/issue-painel-jwt' \
    -H "Authorization: Bearer <USER_JWT>" \
    -H "Content-Type: application/json" \
    -d '{"profile_id": "<UUID>", "scope": "madrinha"}'

Resposta esperada (Madrinha ativa):
  {
    "jwt": "eyJhbGc...",
    "expires_at": "2026-...",
    "scope": "madrinha"
  }

Outras respostas a esperar:
  401 invalid_user            - JWT do usuário inválido
  403 forbidden_other_profile - profile_id != auth.uid()
  403 not_a_madrinha          - cliente não é Madrinha
  403 membership_expired      - membership venceu
