Hari, te mando o estado completo do Clube das Madrinhas — o pivô estratégico
do Cuidado Contínuo que substitui o modelo de 3 tiers de assinatura que tinha
no design anterior. O motivo da mudança e a tese estão no doc 45 do Obsidian
(posso compartilhar separado se quiser).

TL;DR do que mudou:

- Saiu: 3 tiers pagos (Essencial / Cuidado R$ 200/mês / Programa 360° R$ 500/mês)
- Entrou: Clube das Madrinhas (360 vagas, fila viva, acesso por escassez)
         + Benefícios High Value liberados automaticamente por pacote ≥ R$ 10k
         + Storytelling do Instituto Estúdio Mais embutido em todas as superfícies

Nada de assinatura recorrente vendida. Nada de regulação ANS/CFM como plano de
saúde. Só escassez + storytelling + trigger automático em client_packages.

  JOIN packages p ON p.id = cp.package_id
  LEFT JOIN unit_prices up ON up.package_id = cp.package_id AND up.unit_id = cp.unit_id
  WHERE COALESCE(up.custom_price, p.price) >= 10000
    AND cp.status = 'active';


ETAPA 4 · Deploy da edge function issue-painel-jwt
─────────────────────────────────────────────────────────────────
  cd <repo-do-app>
  cp -r ../cuidado-continuo-handoff/edge-functions/issue-painel-jwt \
        supabase/functions/

  # Gera secret 32 bytes random
  PAINEL_JWT_SECRET=$(openssl rand -base64 32)
  echo "$PAINEL_JWT_SECRET"     # ← anota, vai precisar no Flask também
  supabase secrets set PAINEL_JWT_SECRET="$PAINEL_JWT_SECRET"

  supabase functions deploy issue-painel-jwt

Testa:
  curl -X POST 'https://<ref>.supabase.co/functions/v1/issue-painel-jwt' \
    -H "Authorization: Bearer <USER_JWT>" \
    -H "Content-Type: application/json" \
    -d '{"profile_id": "<UUID>", "scope": "madrinha"}'

Resposta esperada (Madrinha ativa):
  {
    "jwt": "eyJhbGc...",
    "expires_at": "2026-...",
    "scope": "madrinha"
  }

Outras respostas a esperar:
  401 invalid_user            - JWT do usuário inválido
  403 forbidden_other_profile - profile_id != auth.uid()
  403 not_a_madrinha          - cliente não é Madrinha
  403 membership_expired      - membership venceu
